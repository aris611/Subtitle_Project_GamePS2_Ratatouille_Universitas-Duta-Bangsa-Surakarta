This repository contains an **unofficial, non-commercial Indonesian subtitle translation patch**
for the PlayStation 2 game **Ratatouille**.

The project was developed as an **academic translation and localization project**
by students of **Universitas Duta Bangsa Surakarta**, focusing on language transfer,
contextual meaning, and subtitle readability in video games.

## Project Scope
- Platform: PlayStation 2 (PCSX2 Emulator)
- Content: Indonesian subtitle translation (text only)
- Type: Fan Translation patch
- Purpose: Educational, research, and portfolio use

## Important Notice
- This repository **DOES NOT contain any game files, ISO images, or proprietary assets**.
- You must legally own the original game to use this patch.
